## ğŸ”‘ KEY FILES EXPLAINED

### app.py (182 lines)
**The main Flask application**
- Initializes Flask and database
- Defines all routes (/, /products, /cart, /admin/*, etc.)
- Handles file uploads for product images
- Manages session-based shopping cart
- Implements admin authentication

Key Routes:
- `GET /` â†’ Home page
- `GET /products` â†’ Products listing
- `GET /product/<id>` â†’ Product detail
- `GET /cart` â†’ Shopping cart
- `POST /add_to_cart/<id>` â†’ Add product
- `GET /custom_order` â†’ Custom order form
- `GET /admin/login` â†’ Admin login
- `GET /admin/dashboard` â†’ Admin dashboard
- `GET /admin/product/new` â†’ Add product
- `GET /admin/product/edit/<id>` â†’ Edit product
- `GET /admin/product/delete/<id>` â†’ Delete product

### models.py (35 lines)
**Database models using SQLAlchemy**
- `Product` model: name, description, price, image, availability, size, color
- `CustomOrder` model: customer info, product preferences, notes
- `Admin` model: username, password_hash

### config.py (18 lines)
**Configuration settings**
- Flask configuration
- Database location and settings
- Upload folder settings
- WhatsApp number
- Admin settings

### style.css (~500 lines)
**Mobile-first responsive CSS**
- Responsive breakpoints: 320px, 640px, 1024px
- Mobile-first approach (base styles are for mobile)
- Media queries for tablet and desktop
- No CSS frameworks, pure CSS3

### Templates (11 HTML files)
Each template extends `base.html` for consistent layout
- Mobile-optimized HTML5
- Semantic markup
- Jinja2 template syntax for dynamic content

## âš™ï¸ GETTING STARTED - STEP BY STEP

### Step 1: Prerequisites
```bash
# Check Python version (need 3.8+)
python --version

# Verify pip is installed
pip --version
```

### Step 2: Install Dependencies
```bash
# From project directory
pip install -r requirements.txt
```

### Step 3: Run Application
```bash
python app.py
# Visit http://localhost:5000
```

### Step 4: Configure
Edit `config.py`:
```python
WHATSAPP_NUMBER = 'your-actual-whatsapp-number'
```

### Step 5: Access Admin
```
URL: http://localhost:5000/admin/login
Username: admin
Password: password  # CHANGE THIS!
```

### Step 6: Add Products
- Login to admin panel
- Click "Add New Product"
- Fill in details and upload image
- Click Save

### Step 7: Test Website
- Visit home page
- View products
- Add to cart
- Click "Order via WhatsApp"

## ğŸ› TROUBLESHOOTING

### Issue: "Module not found"
**Solution:** `pip install -r requirements.txt`

### Issue: "Database error"
**Solution:** Delete `baskets.db` and restart app

### Issue: "Admin login fails"
**Solution:** Check credentials, verify database exists

### Issue: "Images not uploading"
**Solution:** Check `/uploads` folder exists and has write permissions

### Issue: "WhatsApp link doesn't work"
**Solution:** Verify WhatsApp number includes country code

See [TESTING_GUIDE.txt](./TESTING_GUIDE.txt) for more troubleshooting.
